# AI-Powered Expense Tracker with Smart Assistant

## Overview

AI-Powered Expense Tracker is a MERN-stack application designed to help users track expenses, analyze spending habits, and receive AI-driven financial insights.

## Features

### Expense Management

* Add Expense
* View Expenses
* Update Expense
* Delete Expense

### Database

* MongoDB Atlas Integration
* Cloud-based Expense Storage

### Backend APIs

* POST /api/expenses
* GET /api/expenses
* PUT /api/expenses/:id
* DELETE /api/expenses/:id

### Upcoming Features

* Expense Analytics Dashboard
* Category-wise Reports
* Monthly Expense Summary
* AI Smart Assistant
* Savings Recommendations
* Spending Pattern Analysis

## Technology Stack

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

### Frontend (Upcoming)

* React
* Vite
* Axios

### AI Integration (Upcoming)

* OpenAI API
